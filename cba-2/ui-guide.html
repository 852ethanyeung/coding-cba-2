UI Guide

So you don't accidentally download a virus

The Micro:Bit logo in the center of the header is not only decorative, but also serves as a clickable link that will send you to the home page.

When hovered (or focused) it will turn gold.

menu

The menu icon on the bottom left corner of the header opens up a dropdown menu when clicked. It shows the primary navigation menu, i.e. sections of the entire website.

When hovered (or focused) it will turn gold. When browser width exceeds a certain size, it will disappear and an expanded primary navigation menu in the same spot will take its place.

more_vert

The vertical dots on the bottom right corner of the header also opens up a dropdown menu when clicked. It shows the secondary navigation menu, i.e. sections of the current webpage.

When hovered (or focused) it will turn white. When browser width exceeds a certain size, it will disappear and the sidebar to the right of the main content area will take its place.
