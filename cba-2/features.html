Website structure

Every website needs a clear structure. A tool I used for this are semantic tags in the HTML. HTML layout elements I used include:

  • <nav> for the primary and secondary navigation menus
  • <header> for the header section that sticks to the top of the screen as the user scrolls down
  • <main> to contain the main contents, as well as a secondary navigation system in a sidebar
  • <article> to separate the actual content in the <main> tag from the sidebar
  • <aside> for the secondary navigation menu that appears when the screen width is large enough
  • <footer> to give webpages an ending block

Header

Sticky header

To allow the header, which contains the navigation menus, to remain in place at the top of the screen as the user scrolls down, I used position: sticky; and top: 0; to keep the header in place.
