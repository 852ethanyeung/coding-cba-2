General

Website structure

Every website needs a clear structure. A tool I used for this are semantic tags in the HTML. HTML layout elements I used include:

  • <nav> for the primary and secondary navigation menus
  • <header> for the header section that sticks to the top of the screen as the user scrolls down
  • <main> to contain the main contents, as well as a secondary navigation system in a sidebar
  • <article> to separate the actual content in the <main> tag from the sidebar
  • <aside> for the secondary navigation menu that appears when the screen width is large enough
  • <footer> to give webpages an ending block

Favicon

A favicon is a small icon that is displayed next to the page title in its tab. I used the logo as the favicon for the website.

Sticky header

To allow the header, which contains the navigation menus, to remain in place at the top of the screen as the user scrolls down, I used position: sticky; and top: 0; to keep the header in place.

Flexbox layout

In order to make the logo remain at the center on the page, whilst putting the navigation menus on either side, I used display: flex; and flex: 1; to achieve this effect. This is better than giving each elements fixed widths as a flexbox layout would be more flexible and allow the logo to remain centered whatever the viewport width is.

Color change on hover

Using the :hover pseudo-class, I can change the color of the menu icons and the logo when hovered (or focused) to indicate to the user that it is a clickable element. For the logo, as I cannot change the image directly, I instead used background-image: ... which allows me to change it into another image when hovered.

Gradient

To give the header background a gradient effect, I used background: linear-gradient(150deg, black 55%, gold 85%);. This is better than an image of a gradient as it will remainsmooth and unpixelated as the header width grows and shrinks.
